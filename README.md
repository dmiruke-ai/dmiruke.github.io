# dmiruke.dev

Personal portfolio site. Hosted via GitHub Pages from this repo, served at https://dmiruke.dev.

The current home page documents **platform-engineering-lab** — a production-grade reference architecture
for an internal Platform Engineering team. Multi-environment AWS infrastructure via Terraform, a complete
in-cluster platform layer (ingress, observability, security, FinOps, GitOps), and a Phase 4 agentic
layer that generates IaC from intent.
